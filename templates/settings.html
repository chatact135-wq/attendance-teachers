VPN / Proxy Protection

This is a server-side IP check. It can block many VPN/proxy/datacenter IP addresses, but no website can detect every fake GPS app 100%.

For stronger detection, add Railway variable PROXYCHECK_API_KEY. Without it, the app uses a basic free IP check fallback.
